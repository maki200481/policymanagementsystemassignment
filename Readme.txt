Insurance Policy Management System
This system is designed to manage policyholders, products, and payments for an insurance company. It allows policy managers to add and suspend policyholders, manage policy products, and handle payments.
Files Description
- `main.py`: This is the entry point of the application. It demonstrates the functionalities of registering policyholders, processing payments, and managing products.
- `policyholder.py`: Contains the `PolicyHolder` class with methods to register, suspend, and reactivate policyholders.
- `product.py`: Contains the `Product` class with methods to create, update, and suspend products.
- `payment.py`: Contains the `Payment` class with methods to process payments, send reminders, and apply penalties.
